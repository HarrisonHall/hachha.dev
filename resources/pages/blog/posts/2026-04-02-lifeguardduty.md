This is just a small announcement about
[Lifeguard Duty](https://hachha.dev/lifeguard-duty/).

Lifeguard Duty is a simple (buggy) arcade game about keeping a lazy river safe.
The concept was essentially "what if anomaly hunting games weren't scary" +
"arcade games are fun." While I still think this concept has merit, as-is
Lifeguard Duty isn't all that fun. I tried a few variants, but I likely won't
resume work on the gameplay.

As far as development went, this was my first time using
[macroquad](https://github.com/not-fl3/macroquad), which had a great devex.
